## :zap: Installation

1. Clone the Repository.
```bash
    $ git clone https://github.com/thesauravkarmakar/Fake-News-Detection
 ```
2. Install dependencies 
```bash
    $ pip install -r requirements.txt
 ```
3. Run 
```bash
    $ python app.py
 ```
